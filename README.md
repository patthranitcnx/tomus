# Fertilizer CRM

ระบบ CRM สำหรับธุรกิจขายปุ๋ย ที่ช่วยบริหารลูกค้า, บัญชี, และคอมมิชชั่นสำหรับเซลส์

## สถานะปัจจุบัน

- โครงสร้าง Next.js + TypeScript ถูกสร้างขึ้น
- Prisma schema สำหรับฐานข้อมูล CRM ถูกเตรียมไว้
- มีหน้าเริ่มต้นสำหรับลูกค้า, ใบแจ้งหนี้, ทีมขาย และรายงาน

## การติดตั้ง

```bash
npm install
npx prisma generate
npm run dev
```

## จุดเริ่มต้น

- เปิด `/src/app/page.tsx` เพื่อปรับแต่งแดชบอร์ด
- ดู `prisma/schema.prisma` เพื่อเพิ่มโมเดลลูกค้าหรือคอมมิชชั่น
